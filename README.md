# 🚀 MindSpark — Hướng dẫn deploy lên Replit

## Bước 1: Tạo Repl mới
1. Vào **replit.com** → đăng nhập
2. Nhấn **+ Create Repl**
3. Chọn template: **PHP Web Server**
4. Đặt tên: `mindspark` → nhấn **Create Repl**

## Bước 2: Upload files
Xóa file `index.php` mặc định, rồi upload tất cả các file sau:
- index.php
- login.php
- register.php  
- dashboard.php
- chat.php
- flashcard.php
- quiz.php
- notes.php
- planner.php
- logout.php
- db.php
- navbar.php
- ai_api.php
- save_quiz.php
- style.css

## Bước 3: Chạy
Nhấn nút **Run** → Replit sẽ cấp link dạng:
`https://mindspark.yourname.repl.co`

## Bước 4: Dùng thôi! 🎉
- Mở link → Đăng ký tài khoản → Đăng nhập
- Share link cho giáo viên là xong!

## Lưu ý
- Database SQLite tự tạo khi chạy lần đầu
- Groq API key đã được gắn sẵn (14,400 req/ngày)
- Replit free tier: web ngủ sau 30 phút không dùng, tự thức dậy khi có người vào
